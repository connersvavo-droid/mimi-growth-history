# 咪咪成长史 🐱

记录一只猫从2个月到11岁的每一个珍贵瞬间。

## 关于这个项目

这是一个插画线条可爱风格的单页网站，展示咪咪（一只黑猫）的成长历程。

## 功能模块

- **那年今日** — 回顾每年同一天的照片
- **昵称变化史** — 从小咪到性感大黑唇的演变
- **生日记录** — 8月20日，陪伴第11年
- **过节礼物集** — 圣诞节/春节/生日收到的礼物
- **成长时间线** — 横向轮播展示所有照片
- **积分体系** — 两榜综合排名第一可获得「私生饭」称号🏅

## 技术栈

纯 HTML + CSS + JavaScript，无框架依赖。

## 本地预览

直接用浏览器打开 `index-cute.html` 即可。
